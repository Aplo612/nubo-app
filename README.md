# NUBO – Plataforma Gamificada de Reciclaje y Misiones Ambientales

NUBO es una aplicación móvil diseñada para impulsar hábitos sostenibles mediante un sistema de misiones ecológicas, recompensas y contenido educativo interactivo. Su objetivo es convertir acciones cotidianas como reciclar, reducir residuos o participar en actividades ambientales en una experiencia simple, divertida y motivadora.
